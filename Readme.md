AI Voice Assistant - Intelligent Speech Command System 
A sophisticated AI voice assistant that listens to natural speech commands, processes them intelligently, and executes tasks including web automation through Chrome browser control. Built as my project from the Python for Beginners Certification (Alison.com).  

 #Key Features:  
    - Real-Time Speech Recognition - Converts spoken words to actionable text instantly  
    - Natural Language Understanding - Intelligently parses voice commands  
    - Web Browser Automation - Controls Chrome via Selenium WebDriver  
    - Dynamic Search & Navigation - Voice-triggered Google
